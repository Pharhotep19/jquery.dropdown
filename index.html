Examples

Each example below demonstrates various features of the dropdown. The source code can be viewed to see implementation methods.

Form select

Selecting an option will automatically update the original form control and vice-versa

Multi-select

Dropdown will automatically close once the maximum number of items has been selected

By default, <select> elements with the multiple attribute will be converted to a multi-select dropdown.

To explicitally enable or disable, set multi to true or false. Set to null to auto-detect.

Nested

Dropdown position will be remembered when reopened

By default, nested lists and form selects will be converted into nested dropdown menus.

To disable this behaviour, set nested to false.

Grouped (nesting disabled)

An extra divider has been added to seperate Option 4

By default, dividers are automatically added above labels.

They can be explicitally added above and/or below items by setting divider to "top", "bottom" or "both" on the item.

By default, link items will not be selected and instead will be treated as any other link on the page.

To disable this behaviour, set followLinks to false.
